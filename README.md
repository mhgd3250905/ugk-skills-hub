# UGK Skills Hub

集中管理分散在各处的 Agent 技能。

## 目录结构

```
skills/
├── github-helper/     ← GitHub 综合技能（检索 + 管理）
└── ...                ← 更多技能陆续迁入
```

## 技能说明

每个技能目录包含：
- `SKILL.md` — 技能定义和使用说明
- `scripts/` — 执行脚本
- `references/` — 参考文档
- `plans/` — 执行计划（如有）
- `templates/` — 模板文件（如有）
- `evals/` — 测试用例（如有）
