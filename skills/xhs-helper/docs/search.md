# 关键词检索

## 完整流程

```
用户请求 → 执行标准检索脚本 → 读取报告 → 整理总结发给用户
```

### Step 1：执行脚本

```bash
node scripts/xhs-standard-search.mjs \
  --keyword "当阳美食" [--max 20]
```

| 参数 | 说明 |
|------|------|
| `--keyword` | 搜索关键词（必填） |
| `--max` | 最多提取条数，默认 20 |

### Step 2：读取报告

脚本执行完成后在终端输出完整 Markdown 报告，同时保存到 `{关键词}-检索报告.md`（skill 目录下）。

### Step 3：整理总结

agent 读取报告内容，根据用户需要整理成结构化结果发给用户，例如：
- 按分类整理（餐厅推荐 / 特色菜品 / 评价口碑）
- 提取关键信息（名字、地址、推荐理由）

---

## 脚本工作流程（仅用于排查）

1. 打开搜索页 → 筛选图文笔记
2. 滚动加载直至达到目标条数
3. 提取卡片位置坐标
4. 逐一点击卡片 → 等待弹窗加载 → 提取正文 → Escape 关弹窗
5. 生成 Markdown 报告

**关键设计**：只打开一个搜索页面，逐一点击卡片弹窗提取详情，不反复刷新页面。
