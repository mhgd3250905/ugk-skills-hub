# 小红书知识库

收集小红书创作相关的知识和最佳实践。

## 平台特点

### 用户画像
- 主要用户：18-35岁女性为主
- 关注领域：生活方式、旅游、美食、穿搭、护肤、健身
- 内容偏好：真实体验、实用干货、情感共鸣

### 平台文化
- **真实感**：用户喜欢真实的个人体验分享
- **生活化**：接地气的表达，不做作
- **互动性**：评论区活跃，用户喜欢互动
- **视觉化**：图片/视频质量要求高

## 内容创作技巧

### 标题公式

| 公式 | 示例 | 适用场景 |
|------|------|----------|
| 痛点+解决方案 | `皮肤干裂？这个方法拯救你` | 护肤、健康 |
| 地点+体验 | `逃离城市喧嚣｜嵊泗海岛治愈之旅💙` | 旅游 |
| 数字+结果 | `3天瘦5斤｜懒人减肥攻略` | 减肥、健身 |
| 对比+反转 | `后悔没早知道｜原来可以这样` | 知识分享 |
| 时间+成本 | `人均500｜3天玩转XX` | 攻略类 |

### 正文结构

```
【开头】一句话吸睛（痛点/悬念/情感）
    ↓
【中间】核心内容（分点阐述，每点换行）
    ↓  
【结尾】互动引导（提问/推荐/祝福）
```

## 配图技巧

### 图片数量
- 推荐：3-6张
- 最少：1张
- 多：9张（九宫格）

### 图片顺序
1. 首图：最吸引眼球的照片（封面）
2. 中间：核心内容照片
3. 结尾：总结或互动相关照片

### 图片风格

| 风格 | 适合内容 |
|------|----------|
| 光影 | 自然风景、海景、日出日落 |
| 简约 | 美食、日常、产品 |
| 备忘 | 攻略、Tips、清单 |
| 边框 | 人物照、穿搭 |

## 发布时机

### 最佳发布时间（北京时间）

| 时间段 | 适用内容 | 原因 |
|--------|----------|------|
| 7:00-9:00 | 早餐、通勤、正能量 | 早起用户刷手机 |
| 12:00-14:00 | 美食、午餐、休闲 | 午休时间 |
| 18:00-20:00 | 下班、晚餐、穿搭 | 下班高峰 |
| 21:00-23:00 | 深度内容、护肤、情感 | 晚间浏览高峰 |

### 发布频率
- 推荐：每天 1-3 篇
- 最少：每周 3-5 篇
- 注意：不要短时间内发多篇，间隔 2-4 小时

## 风险规避

### 内容审核

| 禁止类型 | 说明 |
|----------|------|
| 广告硬广 | 明显的商业推广会被限流 |
| 敏感话题 | 政治、社会热点谨慎处理 |
| 虚假信息 | 夸大效果、虚假体验会被处罚 |
| 侵权内容 | 未经授权使用他人图片文字 |

### 被限流的原因
- 内容质量差
- 明显广告推广
- 大量复制搬运
- 敏感话题讨论
- 标题党、夸张标题

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 笔记审核不通过 | 检查是否有敏感词、广告词，修改后重新发布 |
| 浏览量很低 | 检查标题吸引力、发布时间、话题标签 |
| 被限流 | 避免广告推广，提高内容质量，减少搬运 |
| 图片上传失败 | 使用文字配图功能，或检查图片格式大小 |
